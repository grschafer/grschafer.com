Tetradrop
(because I heard people get sued for calling these games by the other name)

Author: Greg Schafer
Email: grschafer@gmail.com

To run:
1. Install pygame http://pygame.org/download.shtml
2. Install numpy http://new.scipy.org/download.html
    There should be pygame and numpy folders in your python/Lib/site-packages
    directory.
3. In the src/ folder, run "python tetradrop.py"

What

I got tired of walking across the building over and over one day, always finding the laundry machines taken. So, I decided to set up some Arduinos with a wifi shield and make a website to show laundry machine usage/status and, maybe, mobile notifications for when your laundry finishes. The biggest remaining unknown is how to detect current (or at least, when the machine is running) without spending lots of money on current monitors or other expensive sensors.

Tech

  • Python with Werkzeug to expose RESTful web service endpoints and sqlite3 for data persistence
  • Arduino with WiShield 2.0 and radio transmitter + receiver for communication

When

(Inaccurate estimates for) Progress so far:

  • Web service endpoints for Arduino: 80%
  • Web service endpoints for webpage: 20%
  • Arduino communication (wifi/radio code): 60%
  • Arduino data collection (current monitoring hardware): 0% (currently researching options)

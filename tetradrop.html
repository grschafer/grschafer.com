When Summer 2011
Tech Python with pygame for graphics and user input
What With about a week between the end of my internship and the beginning of school, I decided to practice python and make a game. On the advice of an old gamedev article, I picked Tetris.
How to play
  1. Download the source and extract it
  2. Download and install pygame and numpy
  3. Do python main.py from the src folder to run the game
